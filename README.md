# Los Cosos - Smart Seating Planner

**Los Cosos** is an intelligent web application designed to simplify event seating arrangements. It combines a guest registry, drag-and-drop seating charts, and AI-powered automation to create the perfect layout for your events.

## Features

- **Guest Registry**: Manage your master guest list, track invitations, and categorize guests (Always Invited, Occasional, New, etc.).
- **Smart Seating**: Drag-and-drop interface to assign guests to specific tables and seats.
- **AI Automation**: Uses Google Gemini AI to automatically generate seating plans based on constraints (e.g., keeping couples together, balancing age groups).
- **Visual & List Views**: Toggle between a visual representation of the table (Round, Square, Oval) and a list view.
- **Multi-Device Support**: Optimized for both Desktop and iPhone/Mobile (PWA-ready).
- **History Tracking**: Keep a record of past events and where guests were seated previously.
- **Export & Share**: Download high-resolution images of individual tables to share via WhatsApp.

## Tech Stack

- **Frontend**: React 19, TypeScript, Tailwind CSS
- **AI**: Google GenAI SDK (Gemini 2.5)
- **Icons**: Lucide React
- **Utils**: html2canvas (for image generation)

## Setup

1. Clone the repository.
2. Ensure you have a valid API Key for Google Gemini.
3. Serve the application using a local server (e.g., Vite, or a simple HTTP server) or deploy to a static host.
